# Familiar

**Тихий помощник для macOS, живущий в строке меню.**

Управляет поведением Safari — и задуман как место для других мелких системных удобств.

Familiar работает в фоне: значок в строке меню, без иконки в Dock. Настроил один раз — и забыл.

## Возможности

### 🪟 Окно Safari не закрывается на последней вкладке

Когда в окне осталась одна вкладка и ты жмёшь <kbd>⌘</kbd> + <kbd>W</kbd>, Safari обычно закрывает всё окно. Familiar перехватывает это: открывает новую пустую вкладку и закрывает старую — окно остаётся на месте. Если вкладок несколько, поведение обычное.

### 🔀 Переключение профилей Safari по горячей клавише

Назначь свою комбинацию в настройках и переключайся между профилями Safari одним нажатием, по кругу. Familiar поднимает уже открытое окно нужного профиля (а если такого нет — открывает одно). Профили в Safari недоступны для AppleScript, поэтому переключение работает через Accessibility.

## Установка

Готового бинарника в репозитории нет — `.app` собирается из исходников.

### 1. Сборка

Нужны [XcodeGen](https://github.com/yonaskolb/XcodeGen) и Xcode.

```bash
brew install xcodegen
./build.sh                 # генерирует проект, собирает Release и пакует dist/Familiar.dmg
```

### 2. Установка

1. Открой `dist/Familiar.dmg`, перетащи `Familiar.app` в «Программы».
2. При первом запуске macOS предупредит о непроверенном разработчике (приложение не из App Store) — открой через правый клик → «Открыть».
3. Выдай разрешения, когда система попросит (см. ниже).

Для разработки в Xcode:

```bash
xcodegen generate
open Familiar.xcodeproj
```

## Разрешения

Familiar работает вне песочницы (нужно для перехвата клавиш и управления Safari). При первом запуске система попросит:

- **Универсальный доступ (Accessibility)** — для перехвата <kbd>⌘</kbd> + <kbd>W</kbd> и горячих клавиш, а также для переключения профилей.
- **Автоматизация → Safari** — для управления вкладками.

Без этих разрешений значок в строке меню будет приглушённым, а функции — неактивными.

## Архитектура

| Файл | Назначение |
| --- | --- |
| `main.swift` | Точка входа, режим `.accessory` (только строка меню) |
| `AppDelegate.swift` | Значок, меню, окно настроек |
| `AppState.swift` | Состояние и настройки — единый источник правды |
| `TabGuard.swift` | Перехват ⌘W и управление вкладками Safari |
| `ProfileSwitcher.swift` | Переключение профилей Safari через Accessibility |
| `Hotkey.swift` | Глобальная горячая клавиша (Carbon) |
| `SettingsView.swift` | Окно настроек (SwiftUI) |

Новые функции добавляются такими же отдельными модулями; ядро (`AppState`, `Hotkey`) от Safari не зависит.

## Требования

macOS 13 (Ventura) и новее — из-за `SMAppService` (автозапуск) и профилей Safari.
